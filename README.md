# Asesoría Legal UG 360

Sitio web profesional para **Asesoría Legal UG 360** - Firma legal y notarial especializada en Costa Rica.

## 🎯 Descripción

Página web de servicios legales con portal privado para clientes. Diseñada con una paleta de colores premium (negro, dorado y anaranjado) que transmite profesionalismo y confianza.

## 📋 Características Principales

### Secciones
- **Inicio (Hero)** - Presentación principal con tagline
- **¿Qué es UG 360?** - Descripción de la firma
- **Áreas de Práctica** - Grid de 6 servicios legales
- **Consultoría Online** - Formulario de registro de clientes
- **Portal Privado** - Acceso restringido a expedientes digitales

### Áreas de Práctica
1. Derecho Corporativo
2. Derecho Laboral
3. Derecho Penal
4. Derecho Civil & Mercantil
5. Derecho Inmobiliario
6. Notaría Pública

### Características Técnicas
- Diseño responsive (mobile-first)
- Navegación sticky
- Formulario completo con validación
- Portal privado con tabla de documentos
- Footer con información de contacto
- Animaciones suaves
- Paleta de colores profesional

## 🛠️ Tecnologías

- HTML5
- CSS3 (Grid, Flexbox)
- JavaScript Vanilla
- Font Awesome Icons
- Google Fonts (Segoe UI)

## 🚀 Uso

1. Clonar el repositorio
```bash
git clone https://github.com/asesorialegalug360-dev/asesoria-legal-ug360.git
```

2. Abrir `index.html` en un navegador
```bash
open index.html  # macOS
# o
start index.html  # Windows
```
